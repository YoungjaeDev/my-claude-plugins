## Usage Note

Keep these terms in English when translating.

Good example:
- "Transformer 모델을 Fine-tuning하여..." (O)

Bad example:
- "변환기 모델을 미세조정하여..." (X)

When context requires explanation, add Korean in parentheses:
- "Attention(주의 메커니즘)을 통해..."
