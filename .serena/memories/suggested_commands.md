# Development Commands

## Build & Development
```bash
# Build the project
npm run build

# Watch mode for development
npm run dev

# Start the built application
npm run start
```

## Testing
```bash
# Run tests (watch mode)
npm test

# Run tests once
npm run test:run

# Run tests with UI
npm run test:ui

# Run tests with coverage
npm run test:coverage
```

## Code Quality
```bash
# Lint the source code
npm run lint

# Format the source code
npm run format
```

## Git Commands
```bash
# Check status
git status

# Create a new branch
git checkout -b <branch-name>

# Commit changes
git commit -m "message"

# Push changes
git push
```

## System Utilities (Linux)
```bash
# List files
ls -la

# Find files
find . -name "*.ts"

# Search in files
grep -r "pattern" src/

# Navigate directories
cd <directory>
```

## NPM Package
```bash
# Install dependencies
npm install

# Prepare for publish
npm run prepublishOnly
```
